셀렉터 연습하기

셀렉터 : css에서 html요소를 특정하기위해 사용하는 문법. 1. 전체 셀렉터 2. 태그 셀렉터 3. ID 셀렉터(#) - id속성 값을 지정하여 일치하는 요소들 선택 - 주의사항 : 중복 방지 4. class 셀렉터(.) - class 속성값들을 지정하여 일치하는 요소들을 선택 - 중복을 허용하기 때문에 여러 속성들을 지정할 때 주로 사용. 5. 속성 셀렉터(input태그에서 많이 사용.) 셀렉터[속성] : 지정된 속성을 갖는 모든 요소 선택 셀렉터[속성="값"] : 지정된 속성을 가지며 지정된 값 + 속성 값이 일치하는 모든 요소 선택 셀렉터[속성-="값"] : 지정된 속성 값이 지정된 값을 단어로 포함하는 요소를 선택 셀렉터[속성|="값"] : 속성값이 값과 일치하거나 값으로 시작하는 요소 셀렉터[속성$="값"] : 지정된 속성값으로 시작하는 요소 선택 셀렉터[속성*="값"] : 6. 복합 셀렉터 - 자손 셀렉터 - 자식 셀렉터 - 동위 셀렉터 - 인접 셀렉터 - 일반 셀렉터 7. 의사클래스
